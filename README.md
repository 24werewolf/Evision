build
===
#### MFC
1.OpenCV版本:2.4.13,解决方案文件夹已经内置了一个OpenCV<br>
2.VC14,兼容msvc15-2017,V140.v141工具集<br>
3.需要安装Windows 10 SDK<br>
4.运行exe时若提示缺少某些dll,只需在OpenCV中找到相应的dll放在exe旁边就可以,或者将dll的路径加入到环境变量.<br>
#### QuantEvision
1.Qt版本:Qt 5.9.1<br>
2.Vs版本:2017<br>
